# BRONX - Барбершоп в Шахтах

Лендинг для барбершопа BRONX в Шахтах, разработанный с учетом страховых и возражений целевой мужской аудитории.

## О проекте

Этот проект представляет собой полнофункциональный лендинг для барбершопа BRONX, расположенного в Шахтах. Лендинг разработан с использованием React и Tailwind CSS и включает в себя:

- Адаптивный дизайн для всех устройств
- Анимации при прокрутке
- Форму записи с валидацией
- Информацию об услугах и специалистах
- Отзывы клиентов
- Социальные доказательства в цифрах
- Интерактивную карту проезда

## Улучшения версии 2.0

Были реализованы следующие улучшения для повышения конверсии и улучшения пользовательского опыта:

### 1. Улучшенная Hero-секция
- Добавлены анимированные заголовки с использованием Typewriter Effect
- Внедрены счетчики для отображения статистики (клиенты, опыт, мастера)
- Добавлены интерактивные карточки с преимуществами
- Реализован эффект параллакса для фонового изображения
- Добавлены декоративные элементы для улучшения визуального восприятия

### 2. Оптимизированная секция услуг
- Уменьшена текстовая перегрузка
- Добавлены визуальные акценты (иконки, цветовые выделения)
- Реализовано сравнение с обычной парикмахерской
- Добавлены hover-эффекты для карточек услуг
- Улучшена структура информации для лучшего восприятия

### 3. Интерактивная секция команды
- Добавлена навигация между мастерами
- Реализован переключаемый контент с информацией о каждом специалисте
- Добавлены примеры работ в виде галереи
- Внедрены мини-карточки для быстрой навигации
- Улучшена визуальная составляющая раздела

### 4. Улучшенная секция отзывов
- Добавлены визуальные элементы (аватары, рейтинги)
- Реализована автоматическая смена отзывов
- Добавлены мини-карточки всех отзывов для быстрого доступа
- Улучшена структура отображения отзывов
- Добавлен призыв к действию для оставления отзывов

### 5. Улучшенная форма записи
- Добавлена полная валидация полей
- Реализованы индикаторы загрузки при отправке
- Добавлены сообщения об успешной отправке и об ошибках
- Улучшен UX/UI формы
- Добавлены ограничения на выбор даты и времени

### 6. Новая секция "Социальные доказательства"
- Добавлены счетчики с анимацией (клиенты, рейтинг, гарантия)
- Реализован блок с преимуществами
- Добавлены отзывы клиентов с рейтингами
- Улучшена визуализация статистических данных

### 7. Интерактивная карта проезда
- Внедрена Google Maps с указанием местоположения
- Добавлены кнопки для построения маршрутов
- Реализована информация о парковке
- Добавлены сведения о ближайших объектах
- Улучшена навигационная информация

### 8. Мобильная оптимизация
- Улучшена адаптивность для мобильных устройств
- Оптимизировано меню для мобильной версии
- Улучшена структура блоков для маленьких экранов
- Добавлены мобильные жесты и взаимодействия

## Технологии

- React
- Tailwind CSS
- React Scroll
- AOS (Animate On Scroll)
- React Typewriter Effect
- React CountUp
- Google Maps API

## Запуск проекта

Для запуска проекта в режиме разработки:

```bash
npm start
```

Для сборки production-версии:

```bash
npm run build
```
